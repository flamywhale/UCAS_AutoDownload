# 自动下载UCAS课件

## Attention

**请勿滥用。**
本项目代码仅供学习使用。

## Usage

``` bash
  # python >= 3.8
  pip install -r requirements.txt
  python main.py
```

## Dependency

* 本项目的下载视频部分依赖`youtube-dl`和`ffmpeg`，请自行下载依赖。
    * [youtube-dl](https://ytdl-org.github.io/youtube-dl/index.html)
    * [ffmpeg](https://www.ffmpeg.org/download.html)
